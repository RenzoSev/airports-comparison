# Airports Comparison

For now, this app only compares distance between airports, from the same country.

# IMPORTANT: Running with node version v21.7.1 with PNPM.

# Deployed with Netlify
https://glittery-bublanina-6e1345.netlify.app/

# How to install
```js
nvm install
nvm use
pnpm i
pnpm run dev
```

# Technologies
- PNPM
- React
- TypeScript
- Vite
- SWC
- TailwindCSS
- MaterialUI
- Eslint
